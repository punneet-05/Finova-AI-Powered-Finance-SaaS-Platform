# FINOVA - Full Stack AI Finance Platform

FINOVA is a modern, AI-powered finance tracking platform that helps individuals manage expenses, income, and budgeting—all in one intuitive interface. Built with Next.js and powered by cutting-edge tools, it delivers intelligent insights for smarter financial decisions.

---

## Features

### AI Receipt Scanner

Upload receipts and automatically extract date, amount, description, and category using AI.

### AI-Generated Spending Reports

Get personalized insights and suggestions to optimize spending habits.

### Email Notifications

Receive alerts when spending exceeds your budget.

### Categorization & Filtering

Easily filter and sort transactions by category, type, and date.

---

## Tech Stack

- **Frontend:** Next.js, Tailwind CSS, ShadCN UI
- **Backend:** Next.js API Routes (Serverless)
- **Database:** PostgreSQL (NeonDB)
- **ORM:** Prisma
- **Authentication:** Clerk
- **Email Service:** Resend
- **Security:** Arcjet
- **AI:** Google Gemini AI
